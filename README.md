# Tab Hoarder

Chrome extension for managing browser tabs locally. Like [Toby](https://www.gettoby.com/), but self-hosted and free.

## Planned Features

- Save and organize tabs into collections
- Local-first storage (IndexedDB)
- Optional sync via GitHub Gists or self-hosted server
- Import from Toby export
- Quick search across saved tabs

## Tech Stack

- Chrome Manifest V3
- Vanilla JS or Preact
- IndexedDB for local storage
- Vite for builds
